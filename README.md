# Fountain Cowork Plugins

Plugin marketplace for the Fountain team. Contains skills that power daily Cowork sessions.

## Install

1. Download the installer: [install-fountain-os.command](https://raw.githubusercontent.com/doronstember/fountain-cowork-plugins/main/install-fountain-os.command)
2. Double-click the downloaded file to run it
3. Open a new Cowork session and type `onboard` to get started

If the installer doesn't run, right-click it and choose **Open** (macOS may block it on first launch).

## Plugins

| Plugin | Description |
|---|---|
| **fountain-os** | Triage, reflect, onboard, and task-setup skills for daily work sessions |

## Skills

| Skill | Trigger | What it does |
|---|---|---|
| **onboard** | `onboard`, `set me up` | Sets up a new team member's workspace with personalized CLAUDE.md, memory structure, and task board |
| **triage** | `triage`, `what's going on today` | Morning briefing: calendar, tasks, email, Slack, and team updates |
| **reflect** | `reflect`, end-of-session signals | Reviews the session, proposes memory updates, flags skill candidates |
| **task-setup** | `task-setup` | One-time setup of the task board and sync infrastructure |
