# Fountain Cowork Plugins

Private plugin marketplace for the Fountain team. Contains skills that power daily Cowork sessions.

## Setup

In Cowork, go to **Settings > Plugins > Add Marketplace** and enter:

```
doronstember/fountain-cowork-plugins
```

Then install the **fountain-os** plugin from the marketplace.

## Plugins

| Plugin | Description |
|---|---|
| **fountain-os** | Triage, reflect, onboard, and task-setup skills for daily work sessions |
